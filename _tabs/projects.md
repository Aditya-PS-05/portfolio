### **ProfNITT** • [GitHub](https://github.com/Aditya-PS-05/profnitt) • [Live Demo](https://profnitt.vercel.app/)
An investment and finance club website built with modern web technologies. Features include member management, event tracking, and financial resources.

**Tech Stack:** Next.js, TypeScript, MongoDB, Vercel

![ProfNITT Website](/assets/img/projects/ProfNITT-1.png)
